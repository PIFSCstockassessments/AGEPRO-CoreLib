# AGEPRO

AGEPRO (Age Structured Projection Model) is a program designed to evaluate the likely population consequences of complex harvest scenarios under alternative hypotheses about the stock-recruitment relationship. Uncertainty in initial population size at age is incorporated into the model.

# AGEPRO Core Lib 

AGEPRO __CoreLib__ (Core Library) is an set of object-oriented data structures based on the AGEPRO version 4.0 Input File Structure format. __CoreLib__ includes file handling for the version 4.0 format. 

The purpose for AGEPRO CoreLib is to modularize the GUI and non-GUI functionalities.
